# 造梦西游 Godot 还原项目

目标是在 Godot 4 中逐步还原《造梦西游》1–3 的关卡、角色、战斗、装备和存档系统。
首个里程碑是可独立运行的战斗手感原型，使用原 Flash 的 940×590 逻辑画布；当本地
存在已提取资源时自动显示原作角色姿势，否则使用程序绘制的占位角色。

## 当前可玩内容

- A / D：左右移动
- K：跳跃
- J：普通攻击
- R：重新开始
- 玩家攻击判定、敌人受击/击退、双方生命 UI
- 一代悟空默认装备的待机、移动、跳跃、普攻和受击姿势

## 目录约定

- `scenes/`：Godot 场景
- `src/`：玩法代码
- `assets/original/`：项目原创或可再分发素材
- `assets/extracted/`：从原作提取的本地研究素材（默认忽略）
- `sources/`：来源记录与原始文件工作区

后续优先顺序：素材清点 → 悟空完整动作状态机 → 第一关地图与小怪 → 技能/魔法值 →
掉落、装备和存档，再按同一数据层接入第二、三部内容。
